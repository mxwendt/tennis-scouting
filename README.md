# Tennis Scouting

A tennis match scouting app built with [Elm](https://elm-lang.org) and [Tailwind CSS](https://tailwindcss.com).

## Prerequisites

- [Node.js](https://nodejs.org)

## Install

1. Run `npm install`

## Build

1. Run `npm run build` or run `npm run dist` for an optimized production build

Outputs are written to `dist/`.

## Development

1. Run `npm run dev`
2. Open browser http://127.0.0.1:8080/

## Code Quality

1. Run `npm run format` to auto-format Elm source files
2. Run `npm run review` to lint with elm-review
3. Run `npm test` to execute the test suite
